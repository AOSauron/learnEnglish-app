# Learn'English

**Desktop app for learning English !**

*English Project TELECOM Nancy 2016-2017*

* **Authors**

  - GARCIA Guillaume
  - BLANC Pierre-Arnaud

* **Dependencies**

  - libgl1-mesa-dev
  - qt

* **Build & execute:**

    - `qmake` (if .pro file has been modified, then use make)
    - `make`  (build binary)
    - `./learnEnglish`

    - **Or** simply run `./setup.sh` then `./learnEnglish`
